# Мой.Маршрут

Веб-приложение для управления пассажирскими маршрутами и транспортом (г. Омск, маршрут №212).

Два интерфейса:
- **Водитель** (мобильный) — карта, навигация, смены, отчёты, профиль
- **Предприниматель** (адаптивный) — дашборд, маршруты, транспорт, отчёты, ремонты, зарплата

**Стек:** FastAPI + SQLite + React + TypeScript + Vite + Яндекс.Карты

---

## Требования

- Python 3.10+
- Node.js 18+

---

## Запуск

### Быстрый старт (рекомендуется)

```powershell
cd C:\marshrut
.\start.ps1
```

Скрипт сам установит зависимости, заполнит базу демо-данными и запустит оба сервера.

---

### Ручной запуск

**1. Установить зависимости Python:**
```powershell
cd C:\marshrut
pip install -r requirements.txt
```

**2. Заполнить базу демо-данными** (только при первом запуске):
```powershell
python -m backend.seed
```

**3. Запустить бэкенд** (в отдельном терминале):
```powershell
cd C:\marshrut
uvicorn backend.main:app --reload --port 8000
```

**4. Запустить фронтенд** (в другом терминале):
```powershell
cd C:\marshrut\frontend
npm install   # только при первом запуске
npm run dev
```
